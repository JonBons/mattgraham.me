TAW AdminCP

The AdminCP was created for TAW shortly after an update for our ArmA 2 server came out that essentialy left Linux servers dead in the water.

I set out to create a admin panel that was easy to use and also easy to work with on the backend.

My first decision was to choose a framework that would boost development time. I decided to go with CodeIgniter.

After a few months of development time I ended up with an admin panel that could edit a server's configuration files, view a server's status, view the network graph, view the server logs, and even run RCon commands from the web.

SlickGrid Draggable Grouping

Example

When working with Infragistics I got used to the quick and easy way of dragging column headers on a drop zone and got fustrated with the grouping integration in SlickGrid.

I first started out the project with a SlickGrid plugin but found that the API did not include what I needed to implement a draggable drop zone.

After a little bit of work messing with the drag implementation in SlickGrid I was able to create a simple drop zone that allowed for easy drag and drop grouping of columns..
